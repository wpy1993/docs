# node 相关

## node前端脚本，已大概学会

## node简介
浏览器的JS = es + web api
  web api提供了操作浏览器窗口和页面的能力，BOM、DOM、ajax
  能力有限 - 因为同源策略而存在跨域问题，有限的文件读取能力、无法写入

node中的js = es + node api
  几乎能做java能做的任何事情
  通常用处：
  1. 开发桌面应用程序（eg: VScode）
  2. 开发服务器应用程序
    - 直接操作数据库 - 小型项目适用
    - BFF层，也就是中间层。 同时，把一些轻量级数据和报错，自己维护
  
node强项在于处理大量的吞吐量。java的强项在于强大的计算。所以node适合用来快速转发（BFF）
